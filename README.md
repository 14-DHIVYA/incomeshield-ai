# IncomeShield AI 🚀

## 📌 Problem Statement
Gig delivery workers in India lose income due to external disruptions like heavy rain, extreme heat, pollution, and curfews. Currently, they have no financial protection for such uncontrollable events.

## 💡 Solution
IncomeShield AI is an AI-powered parametric insurance platform that automatically compensates gig workers for income loss caused by external disruptions.

## ⚙️ How It Works
1. Worker registers on platform
2. AI calculates weekly premium based on risk
3. Worker subscribes to policy
4. System monitors weather & disruptions
5. If threshold exceeds → claim auto-triggered
6. Fraud detection checks validity
7. Instant payout processed

## 🤖 AI Features
- Dynamic premium calculation
- Risk prediction based on location & weather
- Fraud detection (anomaly detection, duplicate claims)

## ⚡ Parametric Triggers
- Rainfall > threshold → payout
- AQI > safe limit → payout
- Curfew / shutdown → payout

## 💸 Weekly Pricing Model
Premium is calculated weekly based on:
- Location risk
- Historical disruptions
- Worker’s average income

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB
- AI/ML: Python, TensorFlow
- APIs: Weather API
- Payments: Razorpay (Test Mode)
